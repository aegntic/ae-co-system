SendGrid Setup Instructions

  1. Domain Authentication:

    Add these DNS records to authenticate your domain:

    CNAME em1234.4site.pro sendgrid.net
CNAME s1._domainkey.4site.pro s1.domainkey.u1234.wl.sendgrid.net  
CNAME s2._domainkey.4site.pro s2.domainkey.u1234.wl.sendgrid.net
  2. Create Templates:

    In SendGrid dashboard, go to Email API â†’ Dynamic Templates

    Create each template above with the specified IDs and content

  3. Test Templates:

    Use the SendGrid template testing feature to verify all dynamic data renders correctly

  4. API Integration:

    Use the SendGrid Node.js SDK in your backend:

    const sgMail = require('@sendgrid/mail');
sgMail.setApiKey(process.env.EMAIL_API_KEY);

const msg = {
  to: 'user@example.com',
  from: 'hello@4site.pro',
  templateId: 'welcome_email',
  dynamicTemplateData: {
    name: 'John Doe',
    dashboard_url: 'https://4site.pro/dashboard'
  }
};

await sgMail.send(msg);
