## Animation Parameters

### Timing Sequence
```
Phase 1: Fade In         (0.0s - 2.0s)  - 5 frames @ 400ms each
Phase 2: Heartbeat       (2.0s - 5.0s)  - Loop between frames 6-7 @ 1500ms
Phase 3: Idle            (5.0s - ∞)     - Wait for user interaction
Phase 4: Crumble         (Click + 0.0s) - 4 frames @ 100ms each
Phase 5: Fall            (Click + 0.4s) - 6 frames @ 150ms each
Phase 6: Settle          (Click + 1.3s) - 3 frames @ 100ms each
Phase 7: Typewriter      (Click + 1.6s) - 1 char @ 120ms + random(0-80ms)
```
